# 🎨 UI Design: Best Offer Near Me (Updated)

## 🎯 Design Goal
Create a simple, fast, secure, and attractive app with glass UI and smooth experience.

---

## 📱 App Screens (Phase 1)

### 1️⃣ Splash Screen
- App logo
- App name: Best Offer Near Me
- Simple loading animation

---

### 2️⃣ Login / Signup Screen 🔐
- Email input
- Password input
- Login / Signup button
- Secure authentication (Firebase)

---

### 3️⃣ Home Screen 🏠

#### 🔝 Top Section
- App Name
- Location (Auto-detect 📍)

#### 🔎 Search Bar
- Placeholder: "Search offers, shops..."

#### 🔘 Floating Buttons (Glass UI)
1. 🍔 Food  
2. 👕 Clothes  
3. 🛒 Other  

Features:
- Glass effect (transparent UI)
- Rounded buttons
- Shadow + blur
- Smooth animation

---

### 4️⃣ Result Screen 📜
- List of nearby shops

Each item shows:
- Shop Name
- Offer (e.g., 20% OFF)
- Distance (e.g., 1.2 km)
- Rating ⭐

---

### 5️⃣ Detail Screen 🏪
- Shop full details
- Offer description
- Contact info
- Location on map

---

## 🔐 Security UI Elements

- Login required before access
- Location permission popup
- User-friendly privacy message

- Minimal UI
- Clean layout
